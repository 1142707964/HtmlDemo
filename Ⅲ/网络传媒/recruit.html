市场总监(CMO)

岗位描述 Job Description

  1. 协助总经理制定公司总体发展计划以及战略目标（销售目标＋财务目标）
  2. 为公司提供准确的行业定位，及时提供市场信息反馈
  3. 制定和实施年度市场推广计划和新产品开发计划（依据市场需求的变化，要提出合理化建议）
  4. 依据市场变化要随时调整营销战略与营销战术（包括产品价格的调整等），并组织相关人员接受最新产品知识与市场知识的培训
  5. 制定公司品牌管理与发展策略，维护公司品牌管理、监督和控制公司市场经费用使用情况

岗位要求 Qualifications

  1. 3-8年市场工作经验
  2. 能够独立组织制定市场规划、市场销售策略、产品拓展等工作
  3. 有大型项目的市场拓展和销售工作经验及商务谈判经验
  4. 熟悉互联网，熟练操作办公软件，对教育行业有热情
  5. 独立工作能力和团队合作精神
  6. 具有较强的表达、理解与公关能力
  7. 积极主动、性格开朗、讲效率、乐于接受挑战

市场经理

岗位描述 Job Description

  1. 根据公司整体发展战略及业务发展需求，制定与实施有效的市场推广计划
  2. 负责市场资源开拓，建立有效的传播推广渠道，以及渠道维护，包括包括但不限于网络、展会等常用渠道，线下其他推广渠道
  3. 制定相应的品牌推广计划，提高公司知名度和美誉度，制定有效的品牌提升计划
  4. 公司对外品牌、形象途径的维护与丰富
  5. 参与拟定各种推广活动方案，并负责跟进各种或实施、总结和分析
  6. 建立并维护与公司发展相关媒体、社会组织的关系，为公司发展聚集更多有效资源

岗位要求 Qualifications

  1. 市场营销、广告、传播或相关专业本科以上学历；有优秀的表达能力，亲和力，踏实的工作作风，形象佳，气质好
  2. 互联网及在线教育相关行业工作者；两年以上市场推广工作经验，可独立完成线上、线下推广工作的计划与执行
  3. 策划能力出众，创意新颖，能独立完成中型市场项目方案的策划和提案；有较强的市场感知能力，敏锐把握市场动态、市场方向的能力
  4. 独立组织市场宣传活动，负责网络营销活动的策划、组织及跟踪执行，利用博客、微博、微信等多种平台进行市场及品牌推广
